## Task Descriptions

### Task 4: Network Security Threats Report
Researched and documented four common network security threats: DoS/DDoS attacks, Man-in-the-Middle (MITM) attacks, IP Spoofing, and DNS Poisoning. The report covers how each attack works, real-world examples, impact, and specific mitigation strategies.

**Deliverable:** Research report in Markdown format.

---

### Task 5: Social Engineering Attacks Report
Researched and documented social engineering attack types — phishing (spear phishing, whaling, vishing, smishing), pretexting, baiting, and quid pro quo. The report includes real-world case studies and prevention recommendations for each attack type.

**Deliverable:** Research report with 5-point employee security awareness training checklist.

---

### Task 6: Patch Management Report
Researched the importance of patch management, the vulnerability lifecycle (CVE, CVSS), and real-world breaches caused by unpatched systems (WannaCry, Equifax). The report covers the patch management lifecycle (5 phases), a 7-step best practices checklist, and common challenges.

**Deliverable:** Research report with practical recommendations for organisations.

---

### Task 8: Capture Network Traffic with Wireshark
Captured live network traffic using Wireshark in a safe, isolated lab environment. Applied display filters to isolate HTTP, DNS, and TCP traffic. Identified and annotated a TCP three-way handshake and documented security risks of unencrypted HTTP traffic.

**Deliverable:** PCAP capture file, 4 screenshots, and a detailed README with analysis.
